# Lista de Tarefas

Projeto de gerenciamento de tarefas com Node.js e SQLite.

## Como rodar

1. Clone o repositório.
2. No terminal, instale as dependências:
   npm install
3. Inicie o servidor:
   node server.js
4. Acesse no navegador:
   http://localhost:3000

## Funcionalidades
- Criar, Listar, Editar e Excluir tarefas.
- Reordenar (Arrastar e Soltar).
